For our project we decided to review the website for the company Arrow Disc Distributions. They are a small business here in Sacramento CA, they are a supplier for the popular rec sport disc golf. They sell discs, apparel, and other accessories needed to play the game. We decided to focus on this website because as a small business they likely donâ€™t have a complete understanding of all accessibility guidelines or best practices. Our overall goal for the audit is to make improvements based on the WCAG AA rating. The URL for the website is: https://arrowdiscs.com/ .

Here are some problems and solutions for the audit: +No consistent trace path for tab navigation: -Solution: Maintain a simple left to right trace path for tab navigation +Drop-down menus do not work with the Enter key: -Solution: Modify the script to have the parent menu item act as a toggle to drop down the menu +Missing alt text and empty links: -Solution: Add descriptive alt text and link descriptions for screen readers +Low contrast for text: -Solution: Modify the text to be black for better contrast +Suspicious link and redundant links: -Solution: Modify link text to be more descriptive, disable one of the redundant links +Very small text: -Solution: Increase the size of the font to a more readable size +Shopping cart at the bottom of the page is not a link: -Solution: Make the shopping cart a clickable link +Missing heading levels alert: -Solution: Make a list of items using unordered list code Overall, the main goal of the project was to make improvements based on the WCAG AA rating to improve web accessibility.
